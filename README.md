# SAES Callout System

Игровой MoonLoader-скрипт для отправки каллаутов (запросов поддержки) подразделениям
прямо из SA:MP. Работает в связке с Discord/VK/Telegram-ботом системы SAES.

## Требования

- MoonLoader (0.26+)
- Всё остальное **уже включено** в папку `moonloader/` этого репозитория —
  доустанавливать ничего не нужно, работает даже на урезанной сборке MoonLoader:
  - библиотеки в `moonloader/lib/`: `mimgui`, `effil`, `dkjson`, `encoding` (+`iconv`),
    `socket`/`ltn12`/`mime`, `lfs`, `tabler_icons`;
  - шрифты `Inter-*.ttf` и логотип в `moonloader/resource/fonts/`.
- `ffi` встроен в MoonLoader (LuaJIT) и есть всегда.

## Установка

1. Нажми **Code → Download ZIP** и распакуй архив.
2. Скопируй папку `moonloader` из архива в КОРЕНЬ GTA San Andreas
   (туда, где `gta_sa.exe`). На «Объединить папки?» — Да, на «Заменить файлы?» — НЕТ
   (чтобы не перезаписать уже установленные библиотеки).
3. Перезайди в игру или перезагрузи скрипты (`Ctrl+R`).

## Команды

- `/callout` или `/911` — открыть/закрыть меню
- `/callout update` — обновить скрипт до последней версии

Горячую клавишу открытия можно настроить в меню (значок шестерёнки → «Горячие клавиши»).

## Обновления

Скрипт сам проверяет версию при запуске:

- если доступна новая — покажет уведомление в чате и кнопку «Обновить» в настройках;
- обновиться можно командой `/callout update` (скачает и перезагрузится автоматически)
  или вручную, заменив `moonloader/saes_callout.lua` из этого репозитория;
- если версия слишком старая и больше не поддерживается сервером — потребуется
  обязательное обновление.
